# Asonwabe Ntshiyantshiya - Portfolio Website

A responsive portfolio website showcasing my skills, projects, and experience as an Aspiring Software Developer.

## Features
- Responsive design (mobile-first)
- Clean, modern UI
- Interactive project showcase
- Contact form
- Downloadable resume

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts

## Setup
1. Clone repository
2. Open `index.html` in browser
3. No build process required - it's static!
